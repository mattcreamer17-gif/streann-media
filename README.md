# Streann Media landing page

Bespoke, single-purpose page for Streann Media (booth SMB23, eMerge Americas 2026). The content is the pitch body from `fit-analysis.md` section 7 rendered in Streann's own visual vocabulary: Streann Blue primary, pink ribbon accent, Montserrat throughout, device-native rounded cards with light shadow. Eduba branding is restricted to the footer. The CTA routes to Matt Creamer (CRO) via a real Calendly URL.

Includes an EN/ES toggle because Streann is a Miami-LATAM operator with a bilingual founder team and a customer base spanning 141 countries.

Intended URL path: /for/streann-media
